# String Analyzer in C

A C program that reads multi-line text input and analyzes it.

## 📌 Features

This program counts:

- Vowels
- Consonants
- Digits
- Spaces
- Words
- Lines
- Special Characters

## 🛠 Technologies Used

- C Programming
- Standard Libraries (`stdio.h`, `ctype.h`)

## ▶ How to Compile

```bash
gcc string_analyzer.c
